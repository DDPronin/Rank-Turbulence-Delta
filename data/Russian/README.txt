Full texts of all works used in this study cannot be provided due to copyright restrictions, as the corpus includes contemporary books protected by law.

The folder Russian_sample/ contains some works that are in the public domain and can be freely distributed.

The folder Russian/ contains a metadata table listing all works included in the corpus.

The folder results/ provides the Term-Frequency matrices for all works in the metadata table. These matrices are sufficient to reproduce the experiments.

Text preprocessing procedures are documented in russian_tf.ipynb.
